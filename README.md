# ABVC Training Course

Static course app for American Beach Volleyball Club training videos.

The app lives in `abvc-course/` and can be hosted on any static host. Open `abvc-course/index.html` directly, or serve the folder locally:

```sh
python3 -m http.server 4174 --directory abvc-course
```

Then visit `http://127.0.0.1:4174/`.
